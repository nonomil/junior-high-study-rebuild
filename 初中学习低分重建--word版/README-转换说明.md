# Word文档转换说明

## 关于4个月冲刺版Word文档

由于技术限制，无法直接生成.docx文件。请按以下方法将Markdown文件转换为Word文档：

### 方法一：使用Pandoc（推荐）

1. 安装Pandoc：https://pandoc.org/installing.html
2. 在命令行运行：
```bash
pandoc "14-4个月紧急冲刺版-中考逆袭完整方案.md" -o "14-4个月紧急冲刺版-中考逆袭完整方案.docx"
```

### 方法二：使用在线转换工具

1. 访问 https://cloudconvert.com/md-to-docx
2. 上传 `14-4个月紧急冲刺版-中考逆袭完整方案.md` 文件
3. 点击转换并下载

### 方法三：使用VS Code插件

1. 安装 "Markdown All in One" 或 "Markdown PDF" 插件
2. 打开Markdown文件
3. 使用插件导出为Word格式

### 方法四：复制粘贴到Word

1. 在浏览器中打开 `pages/sprint-4months.html`
2. 全选页面内容（Ctrl+A）
3. 复制（Ctrl+C）
4. 粘贴到Word文档（Ctrl+V）
5. 调整格式

---

## 文件位置

- Markdown源文件：`docs-v2/14-4个月紧急冲刺版-中考逆袭完整方案.md`
- HTML网页版：`pages/sprint-4months.html`

转换完成后，请将生成的Word文档保存到 `初中学习低分重建--word版/` 目录下。
