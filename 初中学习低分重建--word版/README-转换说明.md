# Word 文档说明（docx）

本项目已在 `初中学习低分重建--word版/` 目录下同步提供主要文档的 Word 版本（`.docx`），用于打印或离线阅读。

## 需要自己重新生成/更新 docx？

推荐使用 Pandoc（跨平台、稳定）：

```bash
pandoc "docs-v2/14-4个月稳步提分版-中考上岸方案.md" -o "初中学习低分重建--word版/14-4个月稳步提分版-中考上岸方案.docx"
```

如果你想批量生成，可以对 `docs-v2/*.md` 逐个执行同样的命令，把输出放到 `初中学习低分重建--word版/`。

## 文件位置

- Markdown 源文件：`docs-v2/`
- HTML 网页版：`pages/`
- Word 文档目录：`初中学习低分重建--word版/`
