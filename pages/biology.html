🔄 重点生理过程详解

🌞 光合作用

反应式：

CO₂ + H₂O → (CH₂O) + O₂

条件：

  • 光照（光能）
  • 叶绿体（场所）
  • 叶绿素（催化剂）

过程：

  1. 光反应：在类囊体膜上进行
  2. 暗反应：在叶绿体基质中进行

意义：

  • 制造有机物
  • 释放氧气
  • 转化太阳能

💨 呼吸作用

反应式：

(CH₂O) + O₂ → CO₂ + H₂O + ATP

类型：

  • 有氧呼吸：需要氧气参与
  • 无氧呼吸：不需要氧气参与

过程：

  1. 糖酵解：在细胞质中进行
  2. 柠檬酸循环：在线粒体中进行
  3. 电子传递：在线粒体内膜进行

意义：

  • 分解有机物
  • 释放能量
  • 产生ATP

💧 蒸腾作用

定义：

植物体内的水分以水蒸气的形式从叶片散失到大气中的过程。

过程：

  1. 根部吸收水分
  2. 水分在维管束中运输
  3. 水分到达叶片
  4. 通过气孔散失

意义：

  • 促进水分吸收和运输
  • 促进无机盐的运输
  • 降低植物体温度
  • 参与水循环

🔄 血液循环

循环路径：

  • 体循环：左心室→主动脉→全身→静脉→右心房
  • 肺循环：右心室→肺动脉→肺→肺静脉→左心房

血液成分：

  • 血浆：运输营养物质和废物
  • 红细胞：运输氧气和二氧化碳
  • 白细胞：免疫防御
  • 血小板：止血凝血

功能：

  • 运输营养物质
  • 运输氧气和二氧化碳
  • 运输代谢废物
  • 维持体温

🧬 遗传与变异

基本概念

遗传：

亲代的性状传递给子代的现象。

变异：

子代与亲代之间以及子代个体之间存在差异的现象。

基因：

控制生物性状的遗传因子，是DNA分子上具有遗传信息的片段。

染色体：

细胞核内能被碱性染料染成深色的物质，由DNA和蛋白质组成。

遗传规律

基因的分离定律：

  • 每个性状由一对基因控制
  • 基因在体细胞中成对存在
  • 形成生殖细胞时基因分离
  • 受精时基因重新组合

显性与隐性：

  • 显性基因：用大写字母表示（如A）
  • 隐性基因：用小写字母表示（如a）
  • 显性性状：AA、Aa都表现
  • 隐性性状：只有aa才表现

遗传图解示例：

双眼皮（A）×单眼皮（a）

  • 亲代：AA（双眼皮）× aa（单眼皮）
  • 配子：A × a
  • 子代：Aa（全为双眼皮）
  • 子代自交：Aa × Aa
  • 结果：AA:Aa:aa = 1:2:1
  • 性状比：双眼皮:单眼皮 = 3:1

🔬 重要生物实验

显微镜使用

操作步骤：

  1. 取镜和安放
  2. 对光
  3. 观察
  4. 整理

注意事项：

  • 右手握镜臂，左手托镜座
  • 先用低倍镜观察
  • 调节光线要适中
  • 使用完毕要清洁

观察植物细胞

材料：

  • 洋葱鳞片叶
  • 显微镜
  • 载玻片、盖玻片
  • 碘液

步骤：

  1. 制作临时装片
  2. 染色
  3. 显微镜观察
  4. 绘图记录

探究光合作用

实验设计：

  • 对照实验
  • 控制变量
  • 设置重复

检测方法：

  • 淀粉检测：碘液变蓝
  • 氧气检测：带火星木条复燃
  • 二氧化碳检测：石灰水变浑浊
