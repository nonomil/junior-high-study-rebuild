⚗️ 化学方程式专题突破

📝 方程式书写步骤

  1. 写出反应物和生成物：根据题意确定化学式
  2. 配平化学方程式：使左右两边原子数相等
  3. 标注反应条件：如加热、催化剂等
  4. 标注气体或沉淀：用↑或↓表示

配平方法：

  • 观察法：适用于简单方程式
  • 最小公倍数法：适用于复杂方程式
  • 奇偶配平法：适用于含奇数原子的方程式

🔥 常考方程式汇总

制取氧气：

  • 2KMnO₄ → K₂MnO₄ + MnO₂ + O₂↑
  • 2KClO₃ → 2KCl + 3O₂↑
  • 2H₂O₂ → 2H₂O + O₂↑

燃烧反应：

  • C + O₂ → CO₂
  • S + O₂ → SO₂
  • 4P + 5O₂ → 2P₂O₅
  • 3Fe + 2O₂ → Fe₃O₄

水的反应：

  • 2H₂ + O₂ → 2H₂O
  • 2H₂O → 2H₂ + O₂↑
  • CaO + H₂O → Ca(OH)₂

🧮 化学计算专题

基本计算类型

1. 相对分子质量计算

方法：各原子相对原子质量×原子个数之和

例题：H₂SO₄的相对分子质量 = 1×2 + 32×1 + 16×4 = 98

2. 化学式中元素质量比

方法：各元素质量比 = 相对原子质量×原子个数的比

例题：H₂O中H:O = (1×2):(16×1) = 2:16 = 1:8

3. 元素质量分数

公式：元素质量分数 = (相对原子质量×原子个数)/相对分子质量×100%

解题步骤

  1. 审题：明确题目要求和已知条件
  2. 分析：确定计算类型和使用公式
  3. 列式：根据公式列出计算式
  4. 计算：按步骤进行数值计算
  5. 检验：检查答案是否合理

注意事项：

  • 单位要统一
  • 保留有效数字
  • 结果要合理
  • 过程要完整
