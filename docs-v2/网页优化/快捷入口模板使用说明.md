# 学科页快捷入口模板使用说明

## 1) 模板位置

- HTML 模板：`pages/_templates/quick-jump-panel.html`
- 样式定义：`css/style.css`（已包含 `.quick-jump` 与 `.quick-jump-actions`）

## 2) 适用场景

- 新增任意学科页时，需要在页面顶部给学生一个固定路径：
  - 先做心理建设
  - 再做学习诊断
  - 最后回到当下执行区

## 3) 使用步骤

1. 打开 `pages/_templates/quick-jump-panel.html`。
2. 复制整段 HTML。
3. 粘贴到目标学科页的“心态提示信息块”后面。
4. （可选）在侧边目录里添加：
   - `🌱 每日最低执行线`
   - 锚点建议：`#daily-minimum`

## 4) 默认链接（模板内已配置）

- `mindset-rebuild.html`（心理建设）
- `learning-diagnosis.html`（学习诊断）
- `../index.html#emergency`（当下执行区）

## 5) 常见改法

- 如果页面不在 `pages` 目录同级，按相对路径调整链接。
- 如果不需要 `id="quick-jump"`，可保留 class，删除 id。
- 如果按钮文案想更“鼓励型”，建议只改文字，不改按钮顺序。

## 6) 风格一致性建议

- 快捷入口始终放在学科页开头“心态提示”之后。
- 保持三个按钮顺序一致，避免学生跳转路径混乱。
- 避免添加“必须/立刻/最后机会”等高压词。
